About the course

Wild Code School is a network of schools that train professionals for the most in-demand tech jobs. Their 5-month front-end developer course trains professionals to become web developers. During your training, you will learn multiple skills from 5 main areas (4 technicals and 1 focused on soft-skills)

  • 1/Design a web application
  • 2/ Integrate a user interface
  • 3/ Develop a front-end web application
  • 4/ Perfect yourself as a developer
  • 5/ Become a tech professional
